# cnek .dotfiles

A collection of my personal configuration files for various applications and tools.

#### OS
I use Windows 10, and WSL2 with Ubuntu 20.04 as my primary development environment.

#### Browser
I use Brave as my main web browser, with DuckDuckGo as my search engine.

#### Terminal
I use the zsh shell with oh-my-zsh, config @ [zsh](https://github.com/viktorsynek/.dotfiles/tree/main/zsh) directory.

I also use tmux for terminal multiplexing, config @ [tmux](https://github.com/viktorsynek/.dotfiles/tree/main/tmux) directory.

#### Editor
I use Neovim as my primary text editor, config @ [nvim](https://github.com/viktorsynek/.dotfiles/tree/main/nvim) directory.

#### Window Manager
I use GlazeWM and Zebar as my window manager on Windows, config @ [glaze](https://github.com/viktorsynek/.dotfiles/tree/main/glaze) directory.

#### keyboard
I use ZSA Moonlander, keyboard layout @ [moonlander](https://github.com/viktorsynek/.dotfiles/tree/main/moonlander) directory.
